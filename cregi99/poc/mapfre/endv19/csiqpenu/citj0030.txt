+  CD19002   Quick edit command CCOLS
+
+ This enhancement allows you to use the CCOLS command, or it's
+ abbreviation "CC". You can do this either on the main Endevor screen,
+ or in the element selection panel.
+
+ After invoking the CCOLS command, the user is presented with an ISPF edit
+ session, in which the Element selection list columns can be adjusted.
+ The user can change the column width, the column order, and the color of
+ the column content.
+
+ When you have finished customizing the element selection panel appearence,
+  the changes are saved in the ISPF profile library.
+
