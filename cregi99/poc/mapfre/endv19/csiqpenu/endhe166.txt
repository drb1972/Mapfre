+The following primary commands can be used within the Quick Edit
+Element Selection List to navigate, search and sort the list:
+
% LOCATE element-name+
+    Scroll the list to the element name specified as the command parameter.
+    If the element is not on the list, scroll to the last element name
+    that precedes the specified parameter.
+
% EONLY column-name value+
+    Filters the selection list based upon a specific column,
+    by including or excluding the rows that match your selection criteria.
+
% ESORT [-]column-name+
+    Sorts the selection list based upon a specific column in ascending
+    order. Inserting the "-" sign before column name results in
+    descending order of the sort.
+
% ESEARCH string+
+    Filter the list by searching for the specified string in all elements
+    within the list and removing any elements that do not contain
+    the search string.
+    Using the command without a parameter will open a dialog which allows
+    for more complex searches.
+    This command must be enabled in ENCOPTBL. It is currently:%&DISFTXTS+
+
+ REFRESH+
+    Rebuild the element selection list, removing any applied filters and
+    picking up any changes to the elements since the list was built.
+
+
