/* This panel shopws how to code a vertical scrolling              */
/* section in a panel. See the following:                          */
/* - Attribute byte '$'                                            */
/* - Section )AREA SAREA                                           */
/* - Field SAREA                                                   */
/*                                                                 */
/* Notes (based on observation):                                   */
/* - The line starting $SAREA defines the placement of the scroll  */
/*   area in the )BODY section of the panel                        */
/*   - Multiple lines can be defined with the name (SAREA) only on */
/*     the first line
/*   - Fields on the panel after the scroll area will not display  */
/*     if line $SAREA is defined as a single line as in this panel */
/*   - The dollar sign ($) is a delimiter, not an attribute byte   */
/*     and multiple lines can be defined using it, which allows    */
/*     additonal panel fields and lines to be defined after it     */
/*   - The 2 dollar signs on the line are required for defining    */
/*     the width of the scroll area                                */
/*   - Minimum size of the scroll area is 2 lines                  */
/*   - A "More: - +" display field is automatically generated on   */
/*     the first line of the scroll area                           */
/* - The )AREA section defines the scrolling area                  */
/*   - This area is not limited vertically by the size of the      */
/*     panel or the size of the scrolling area defined in the      */
/*     )BODY section of the panel because it scrolls!              */
/*   - The width of the lines in this section should be no wider   */
/*     than the placement of the dollar sign delimiters in the     */
/*     )BODY section of the panel                                  */
/*   - The dollar signs do NOT use up a byte on the panel lines    */
/*     that they define, they are only delimiters                  */
/*   - Like ISPF table display panels, the scrolling area can      */
/*     vary with the vertical size of the screen when EXTEND(ON)   */
/*     is used                                                     */
/*                                                                 */
